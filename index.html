How to use?

You can download the source code here and compile yourself and make a reference Or you can run the following command from your nuget package manager console:

PM> Install-Package CreativeXamlToolkit

then,

open up your Xaml file, add the following line into your reference:

xmlns:creative="http://creativexamltoolkit.net/winfx/xaml/presentation"

lastly, You can download the source code here and see how the code works in the demo application :-)
